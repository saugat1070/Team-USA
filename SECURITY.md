# Security Policy

## Supported Versions

The following table outlines which versions of **Paila** are currently supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |
| 0.x.x   | ❌ No     |

Only the latest stable release is actively maintained. Users are strongly encouraged to upgrade to the most recent version to receive security fixes.

---

## Reporting a Vulnerability

If you discover a security vulnerability in Paila, please report it responsibly.

### How to Report
- **Email:** saugatgiri1070@gmail.com
- **GitHub:** saugat1070

Please include:
- A clear description of the issue
- Steps to reproduce the vulnerability
- Potential impact
- Any relevant logs, screenshots, or proof of concept

### What to Expect
- You will receive an acknowledgment within **48 hours**
- We will investigate and validate the report
- If accepted, we will work on a fix and coordinate a responsible disclosure
- If declined, we will provide a brief explanation

Please do **not** disclose security issues publicly until they have been addressed.

Thank you for helping keep Paila and its community secure.
